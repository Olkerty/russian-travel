Проект - путешестиве по России
Путешествие по России это учебный проект, состоящий из описания достопримечательностей, которые можно посетить не покидая границ страны и ссылок на ресурсе с дополнительной информацией о них. При создании проекта были использованы html и css для непосредственной вёрстки, git для контроля версий и figmа для работы с дизайнерским макетом. 
GitHub - https://github.com/Olkerty/russian-travel
